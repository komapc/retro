Each cell = one source × event pair

— Dash = searched, nothing found
· Dot = not searched yet
Cell colour = predicted direction
Green — bullish / positive outcome predicted
Red — bearish / negative outcome predicted
Grey — neutral / ambiguous
FALSE badge shown only when event did not happen
Stance scale (−1 to +1)
+1.0 — strongly predicts positive/bullish outcome
0.0 — neutral, no directional view
−1.0 — strongly predicts negative/bearish outcome
